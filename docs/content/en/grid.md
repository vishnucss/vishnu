## Install

Add `@vishnucss/grid` dependency to your project:

<code-group>
  <code-block label="YARN" active>

```bash
yarn add @vishnucss/grid
```

  </code-block>
  <code-block label="CDN">

```html
<link rel="stylesheet" href="https://unpkg.com/@vishnucss/grid" />
```

  </code-block>
</code-group>
