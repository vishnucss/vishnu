## Install

Add `@vishnucss/collapse` dependency to your project:

<code-group>
  <code-block label="YARN" active>

  ```bash
  yarn add @vishnucss/collapse
  ```

  </code-block>
  <code-block label="CDN">

  ```html
<link rel="stylesheet" href="https://unpkg.com/@vishnucss/collapse"/>
  ```

  </code-block>
</code-group>
