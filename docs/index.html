Vishnu

A simple and minimalist toolkit with modern css
Star Currently v1.0.4

Why should I use it?

A simple style guide provides a setup of styles using Modern CSS for a fast and clean starting point. Everything you need to create a basic project is included.

# Install

You can install Vishnu in multiple ways, but it's as simple as adding a single link tag to your project.

CDN
<link rel="stylesheet" href="https://unpkg.com/@vishnucss/vishnu"/>
NPM
$ npm install @vishnucss/vishnu

# Typhography

Based on rem units. All rem units are converted to base 10, so 1rem is equal to 10px. By default, all headings, paragraphs, and code use system fonts, using the most modern one available on a user's operating system.

# Buttons

The Button, an essential part of any user experience. Buttons come in some basic styles: The <button>Content</button> element has default color, whereas .outline has a simple outline around, .flat is entirely clear and .rounded is rounded border radius.

# Tables

The Table element represents data in two dimensions or more. We encourage the use of proper formatting with <thead> and <tbody> to create a <table>. The code becomes cleaner without disturbing understanding.

Add the class .table to any <table> element and .responsive to table responsive.

# Grid

Grid based on the flex display property.

CDN
<link rel="stylesheet" href="https://unpkg.com/@vishnucss/grid"/>
NPM
$ npm install @vishnucss/grid
Responsive

Responsive modifiers enable specifying different column sizes, offsets, alignment and distribution at xs, sm, md & lg viewport widths.

Fluid

Percent based widths allow fluid resizing of columns and rows.

Offsets

Offset a column

Auto Width

Add any number of auto sizing columns to a row. Let the grid figure it out.

# Utils

Provides some basic utility classes to speed up and improve your development.

Installation

CDN
<link rel="stylesheet" href="https://unpkg.com/@vishnucss/utils"/>
NPM
$ npm install @vishnucss/utils
  • pull-left - float left
  • pull-right - float right
  • text-center - center align text
  • text-left - left align text
  • text-right - right align text
  • center - align items within element in the center
  • horizontal-align - horizontally align an element
  • vertical-align - vertical align an element
  • left - align children to the left
  • right - align children to the right
  • center - align children to the center
  • full-screen - make element full screen
  • full-width - make element full width size
  • full-height - make element full height size
  • fixed - make element full width and fixed
  • relative - make element relative
  • truncate - make a single straight line with ellipsis
  • hide-phone - hide elements on a phone
  • hide-tablet - hide elements on a tablet
  • hide-desktop - hide elements on a desktop
